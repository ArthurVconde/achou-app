# achou-app
Aplicativo de lista telefônica digital desenvolvido como projeto acadêmico, com foco em facilitar o acesso a serviços e estabelecimentos locais. O projeto foi construído utilizando plataforma no-code (Base44), com integração à Google Places API para coleta e tratamento de dados. Acesse o app: https://lista-tele-b22b04b5.base44.app
